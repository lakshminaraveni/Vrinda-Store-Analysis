# Vrinda-Store-Analysis

Overview: 
This project involves analysis of Vrinda store using Excel. The goal is to extract valuable insights and answer various business questions based on the dataset. The following README provides a detailed account of the project's objectives, business problems, solutions, findings, and conclusions.

Objective:
Vrinda store want to create and Annual sales report for 2022. So that, Vrinda can understand their customers and grow more sales in 2023.

Business Problems and Solutions:
1. Which month got the highest sales and orders?
Ans: March

2. Who purchased more - men or women in 2022?
Ans: Women

3. What are different order status in 2022?
Ans: Delivered

4. List top 3 states contributing to the sales in 2022?
Ans: Maharashtra, Karnataka and UP

5. Which channel is contributing max to sales?
Ans: Amazon
